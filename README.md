# Tennis Academy Bot Showcase

Public-safe showcase of a Telegram bot for a tennis academy. The bot supports court booking, training registration, tournament flows, and admin notifications.

## Highlights

- Telegram-first booking experience.
- Court availability and booking validation.
- Training and tournament registration scenarios.
- MySQL-backed state in the private implementation.
- Admin notification flow for operational follow-up.

## Tech Stack

- Node.js, Telegraf.
- MySQL, date/time utilities.
- Environment-based configuration.

## Repository Contents

- `docs/architecture.md` - bot flow and service boundaries.
- `examples/booking-flow/bookingValidator.ts` - safe booking validation example.

## Confidentiality Notice

The production implementation is private. This showcase excludes real bot tokens, academy data, customer records, admin chat IDs, and payment provider details.
